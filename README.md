To run these notebooks during the practicals:

- download the repository to your machine (green Code button > Download Zip)
- unzip the folder
- upload the folder to your Google Drive (recommended location: My Drive, drag and drop the folder)
- double click one of the notebooks from Google drive.

If double-clicking a notebook doesn't open it in Google Colab, you may have to right click > open with > Google Colaboratory the first time.
Any browser should work but Chrome is recommended for best compatibility.
